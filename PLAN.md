# 任务岛 v1 产品文档与技术概览

## Summary

任务岛是一款本地优先的 macOS 桌面任务焦点工具。它不是完整项目管理平台，而是一个常驻在桌面顶部的轻量任务层：用液态玻璃悬浮岛持续显示优先级压力、当前焦点、提醒和专注计时；用任务面板承载完整编辑、筛选、设置和数据管理。

当前 v1 已经从最初的“菜单栏 + 胶囊待办”扩展为可日常使用的本地任务工具，支持优先级、自然语言快速新增、任意日期提醒、重复任务、标签/项目、子任务、专注计时、Apple 提醒事项导入导出、外观自定义和安装包分发。

## 产品定位

- **一句话定位**：让重要的事始终在眼前。
- **目标用户**：在 Mac 上长时间工作的个人用户，需要快速捕获任务、保持桌面聚焦、减少切换完整待办 App 的次数。
- **核心场景**：
  - 想到任务时用全局快捷键快速记录。
  - 工作时从顶部悬浮岛看到任务压力和当前专注任务。
  - 需要完整整理时点击悬浮岛打开任务面板。
  - 到提醒时间或进入专注模式时，用中等悬浮岛持续提示。

## 当前功能

### 1. 悬浮岛

- 小胶囊收起态：只显示高/中/低优先级任务数量。
- 中等胶囊专注/提醒态：显示任务标题、状态、倒计时，以及暂停/继续、停止按钮。
- 展开态：悬停后显示最多 3 条重点任务，支持快速新增、固定展开、完成和删除。
- 支持拖拽摆放，默认贴近屏幕顶部，位置可记忆。
- 支持液态玻璃视觉、暗夜玻璃模式、自定义透明度、背景色和文字色。

### 2. 任务面板

- 点击悬浮岛打开完整任务面板。
- 默认显示所有未完成任务，避免“没有放进今天就看不见”的错觉。
- 顶部显示当前任务/专注中任务，支持开始/暂停和停止专注。
- 支持快速新增、搜索、视图切换、面板固定和关闭。
- 任务按高/中/低优先级分组展示。
- 每条任务支持展开详情、开始专注、修改优先级、完成、删除。

### 3. 任务详情

- 标题可直接编辑，回车或点击空白处结束编辑。
- 支持备注、截止时间、提醒时间、重复规则、项目、标签、子任务。
- 支持预计专注分钟，任务专注时长优先使用任务自己的设置。
- 支持快速推迟：稍后、今天晚点、明天、本周。
- 支持“设为当前”，把任务固定到顶部专注栏和菜单栏标题里。

### 4. 当前任务与专注

- 当前任务是用户主动指定的焦点任务，不再作为自动轮转按钮放在专注控制区。
- 开始专注会自动把该任务设为当前任务。
- 专注中显示中等悬浮岛，包含任务标题、倒计时、暂停/继续和停止。
- 停止会退出专注状态；暂停会保留当前专注任务和已累计时间。
- 默认专注时长可在设置中调整，范围 5-180 分钟。

### 5. 快速新增

- 默认全局快捷键：`Control + Option + N`。
- 快捷键可在设置中自定义，支持常见修饰键组合和字母/空格。
- 支持自然语言输入，例如：`明天 10点 发周报 #工作 !高 /30m`。
- 可解析日期、时间、优先级、标签、项目、预计专注分钟和重复规则。
- 快速新增面板支持关闭按钮和 `Esc` 取消。

### 6. 视图与筛选

- 全部
- 今天
- 建议
- 高优
- 即将
- 无日期
- 标签
- 项目
- 已完成
- 回顾

### 7. 系统集成与数据

- SwiftData 本地存储，无账号、无云同步依赖。
- 支持 JSON、Markdown、CSV 导入导出。
- 支持 Todoist 风格 CSV 导入。
- 支持 Apple 提醒事项导入/导出。
- 支持本地通知，提醒到期后触发悬浮岛提醒动画和系统通知。
- 支持 `taskisland://` URL Scheme：
  - `add`
  - `focus`
  - `complete`
  - `show`

### 8. 安装与分发

- 提供 `.app`、`.pkg`、`.dmg` 打包脚本。
- `.pkg` 会安装到 `/Applications/任务岛.app`，注册 LaunchServices / Spotlight，并在安装后启动。
- `.dmg` 提供拖拽到 Applications 的安装体验。
- 当前构建使用本地/ad-hoc 签名，尚未接入 Apple Developer ID 正式签名和公证。

### 9. 品牌与展示素材

- 应用名：任务岛。
- 图标使用浅色、玻璃质感的小岛形态。
- GitHub 展示素材包括 16:9 和 3:4 海报，以及多张功能截图。
- 海报标题：`让重要的事，始终在眼前`。

## 技术架构

- Swift 6.2 + SwiftUI + AppKit。
- macOS 26+，使用 SwiftData 做本地持久化。
- `TaskIslandCore`：任务模型、解析、排序、导入导出、检查逻辑。
- `TaskIsland`：App 生命周期、悬浮岛、任务面板、快捷键、通知、Apple 提醒事项桥接。
- `TaskIslandChecks`：轻量检查脚本，覆盖核心任务规则和导入导出。
- `NSPanel` 用于顶部悬浮岛、任务面板和快速新增面板。
- Carbon hot key API 用于全局快捷键注册。

## 已验证能力

- 任务新增、编辑、完成、删除。
- 优先级统计和排序。
- 日期、时间、重复和快速新增解析。
- 当前任务、专注开始/暂停/停止和专注累计时间。
- 子任务、推迟、标签、项目。
- JSON、Markdown、CSV、Todoist CSV 导入导出。
- Apple 提醒事项导入导出基础流程。
- 打包 `.app`、`.pkg`、`.dmg`。
- GitHub CI 运行 `swift build` 和 `TaskIslandChecks`。

## 仍需注意

- 当前版本未正式签名和公证，跨机器分发时仍可能触发 macOS 安全提示。
- 当前任务机制已经具备基础价值，但未来可以继续强化为“桌面焦点任务”工作流，例如当前任务常驻中等岛、完成后推荐下一项、每日回顾里关联当前任务。
- 目前不做账号、云同步、团队协作、看板、Gantt 或完整日历。

## 下一阶段建议

1. 当前任务增强：不专注时也可选择让当前任务常驻中等悬浮岛。
2. 专注完成后的轻量流程：完成、继续、休息、选择下一个当前任务。
3. 每日回顾增强：完成任务、推迟任务、专注时长和明日建议。
4. 正式签名与公证：Developer ID Application / Installer 证书和 Apple Notary Service。
5. 更细的提醒体验：到期后在悬浮岛中提供稍后提醒、完成、打开详情。
