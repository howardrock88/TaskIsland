TaskIsland Privacy Policy

Last updated: June 13, 2026

Data handling

TaskIsland is a local-first macOS task manager. The current version does not require account login, does not provide cloud sync, does not include advertising SDKs, and does not use third-party analytics. Tasks, tags, projects, reminder times, and focus records are stored locally on the user's Mac by default.

Apple Reminders permission

If the user grants access to Apple Reminders, TaskIsland uses that access only when the user explicitly imports from Apple Reminders or exports TaskIsland tasks to Apple Reminders. TaskIsland does not upload reminders content to developer servers.

Local notifications and files

TaskIsland may request local notification permission to show reminders configured by the user. Users may also explicitly import or export JSON, Markdown, and CSV files.

Contact

If you have questions, please contact the developer through GitHub Issues.
