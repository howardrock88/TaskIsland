
## 审核备注摘要

```text
TaskIsland is a local-first macOS task manager. Task data is stored locally by default. Apple Reminders access is only used when users explicitly import from or export to Apple Reminders. Notifications are used only for local task reminders. File access is used only for user-selected import and export files. The global shortcut only opens the Quick Add panel and does not record keyboard input. The Mac App Store package does not install a LaunchAgent.
```
