# App Store 年龄分级填写建议

当前版本是本地任务管理工具，不包含社交、浏览器、用户生成内容、广告、博彩、医疗、成人内容或暴力内容。按当前功能，年龄分级通常可按 `4+` 方向填写。

在 App Store Connect 的 Age Rating 问卷中，建议按下面填写。最终仍以你提交时页面展示的问题为准。

## 内容频率

```text
Cartoon or Fantasy Violence: None
Realistic Violence: None
Prolonged Graphic or Sadistic Realistic Violence: None
Profanity or Crude Humor: None
Mature or Suggestive Themes: None
Horror/Fear Themes: None
Medical/Treatment Information: None
Alcohol, Tobacco, or Drug Use or References: None
Simulated Gambling: None
Sexual Content or Nudity: None
Graphic Sexual Content and Nudity: None
```

## 交互能力

```text
Unrestricted Web Access: No
Gambling and Contests: No
User-Generated Content: No
Messaging and Chat: No
Location: No
Advertising: No
```

## 解释口径

如果页面要求解释，可以使用：

```text
TaskIsland is a local-first macOS productivity app for tasks, reminders, and focus timing. It does not provide social networking, user-generated content, unrestricted web browsing, gambling, advertising, or mature content.
```

## 上线前复核

如果未来加入这些功能，需要重新评估年龄分级：

- 内置网页浏览器或任意网页访问
- 社区、评论、共享内容或消息聊天
- 广告
- 外部内容源
- 位置能力
- 付费抽奖、博彩或竞赛
