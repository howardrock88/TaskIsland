## Description

TaskIsland is a local-first floating task manager for macOS. Instead of hiding your to-dos in another window, it keeps your most important work visible in a lightweight island at the top of the screen.

The app supports three island states for different moments: a compact number island for task counts, a focus island for the current task and countdown, and an expanded action island for reviewing and handling priority tasks.

You can quickly add tasks, set priority, reminders, due dates, tags, projects, and focus minutes. When you start a focus session, TaskIsland shows the current task and countdown in the floating island, with pause and stop controls within reach. Task data is stored locally by default.

Key features:

- Number island: See high, medium, and low priority task counts at a glance.
- Focus island: Keep the active task, countdown, pause, and stop controls visible.
- Action island: Expand the island to review key tasks and quickly add, complete, or delete items.
- Task panel: Browse all tasks, today, high priority, upcoming, no date, tags, projects, and completed views.
- Task details: Add notes, reminders, due dates, repeat settings, tags, projects, and focus minutes.
- Appearance settings: Customize island opacity, background color, text color, and priority colors.
- Chinese and English interface: Switch UI language in settings without translating user-entered task content.
- Local-first data: Tasks are stored on the Mac, with JSON, Markdown, and CSV import/export.
- System integration: Supports local notifications, quick add, and Apple Reminders import/export.

TaskIsland is for people who want the current task to stay visible, reduce window switching, and keep a light, local-first workflow.

## What's New
